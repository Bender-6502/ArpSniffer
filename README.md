# ArpSniffer

## Linux specific project in C++/CMake

Written out of necessity as my new network router appears to be blocking arp packets to my devices connected to my router's WiFi, while my backhaul ap appears fine?
The mind boggles!

There's probably 100s of examples of this, but where's the fun in using someone elses work when I can experiment myself :)

Supports filtering, for more help try the following:
```
./ArpSniffer --help
```
